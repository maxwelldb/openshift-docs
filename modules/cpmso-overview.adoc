= Control Plane Machine Set Operator overview

[role="_abstract"]
You can use the Control Plane Machine Set Operator to automate management of control plane machines in your {product-title} cluster.
The Operator can replace degraded machines and roll out configuration changes automatically.

When you set the control plane machine set state to `Active`, the Operator reconciles the cluster to the correct number of control plane machines.
It applies the configuration from the custom resource.
That state allows automated replacement of degraded control plane machines and rollout of changes to the control plane.

A cluster has only one control plane machine set, and the Operator only manages objects in the `openshift-machine-api` namespace.

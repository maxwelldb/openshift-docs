= Removing imagestreamtags

You can remove old tags from an imagestream.

.Procedure

* Remove old tags from an imagestream:
+
----
$ oc tag -d <image-name:tag>
----
+
For example:
+
----
$ oc tag -d python:3.5

Deleted tag default/python:3.5.
----
